There Is and There Are

Граматика → VERB → There Is and There Are

1Пояснення теми

Конструкції there is та there are використовуються в англійській мові для вираження існування або наявності когось/чогось у певному місці. На відміну від української мови, де ми кажемо просто «На столі книга», англійська вимагає формальної конструкції з підметом there.

Ця конструкція має свої граматичні особливості: узгодження дієслова з наступним іменником, особливості вживання в заперечних та питальних реченнях, а також можливість поєднання з різними часами (Past, Future, Perfect). Помилки в цій темі є одними з найпоширеніших серед українських учнів.

2Правила та приклади використання

2.1 There is / There are у Present

Утворення:

  • There is + однина / незлічуваний іменник
  • There are + множина
There is a book on the table.
На столі є книга.
There are many students in the classroom.
У класі багато учнів.
There is some water in the bottle. (незлічуване)
У пляшці є трохи води.

2.2 Заперечення та питання

Заперечення: дієслово be + not (isn't / aren't)

Питання: інверсія — Be + there...?

There isn't any coffee left.
Кави не залишилося.
Are there any questions?
Чи є запитання?
Is there a bank near here?
Чи є тут поблизу банк?

2.3 There is/are в інших часах

Past Simple: there was / there were

Future Simple: there will be

Present Perfect: there has been (однина) / there have been (множина)

Be going to: there is/are going to be

There was a concert here last night.
Тут учора ввечері був концерт.
There will be a meeting tomorrow.
Завтра буде зустріч.
There have been several accidents on this road.
На цій дорозі сталося кілька аварій.

2.4 There seems to be / There used to be

There seems/appears to be — здається, що є... (узгодження з there — завжди однина)

There used to be — колись був/були...

There seems to be a problem with the engine.
Здається, є проблема з двигуном.
There used to be a cinema in our town.
У нашому місті колись був кінотеатр.

3Особливості та типові помилки

  • There is/are vs It is/They are: There is/are — про існування/наявність; It is/They are — про ідентифікацію чи опис. It is a book on the table ❌ → There is a book on the table ✅
  • Незлічувані іменники: завжди вживаються з there is: There is some advice / information / news / furniture / traffic.
  • Some/any: some — у ствердженнях; any — у запереченнях та питаннях. There are some apples. Is there any milk? There isn't any sugar.
  • Proximity rule (правило близькості): якщо після there is/are йде кілька іменників, дієслово часто узгоджується з найближчим: There is a pen and two books. Проте в сучасній англійській краще уникати цієї конструкції або використовувати множину, якщо більшість елементів — множинні.
  • Числівники: після there is/are числівник визначає форму дієслова. Але з незлічуваними сумами/кількостями: There is a lot of / lots of / plenty of — форма залежить від наступного іменника.

4Вправа: текст із пропусками

There (be) a beautiful park near my house. There (be) many flowers in the garden. (be) there any milk in the fridge? There (be) a concert here last Sunday. There (be) going to be a test next week.

5Вправа: оберіть правильну форму

1. There ___ a lot of traffic today.
2. ___ any messages for me?
3. There ___ some interesting news in the paper.
4. There ___ a pen, a notebook and two books in my bag.
5. There ___ many people at the concert last night.

6Вправа: оберіть правильну відповідь

1. There ___ no water left in the bottle.
2. ___ there a bank near here?
3. There ___ been several accidents on this road.
4. There ___ to be a problem with the engine.
5. There ___ some cake left if you want it.

7Вправа: перекладіть речення

1. На столі є дві книги.
2. Чи є поблизу аптека?
3. У кімнаті було багато людей.
4. Завтра буде зустріч.
5. У холодильнику є трохи молока.

8Вправа: виправте помилки або оберіть правильний варіант

1. ___ a beautiful park in our city.
2. There ___ some apples on the table.
3. There ___ a lot of snow last winter.
4. There ___ any sugar in my tea.
5. There ___ many reasons to learn English.
